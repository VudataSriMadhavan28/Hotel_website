# Hotel_website
## Royal Stays
A modern, responsive, and user-friendly hotel website developed as part of the MANTRA 2026 Summer School - Frontend Website Assignment. This project showcases a luxury hotel's online presence with elegant design, room information, image gallery, and an online booking form.


## Project Overview

The Royal Stay Hotel Website is a static frontend project built using HTML, CSS, and JavaScript. The website is designed to provide visitors with information about the hotel, its facilities, room categories, and booking services while ensuring a smooth experience across desktop, tablet, and mobile devices.

This project focuses on:

Responsive web design
Clean and modern UI/UX
Basic Search Engine Optimization (SEO)
Frontend development best practices
Website deployment using Netlify
